# IAO
IAO 문제 번역

## 파일이름
``<YYYY>_IAO_<A>_<B><C>.tex``
* ``<YYYY>``: 연도 (예시 ``2008``)
* ``<A>``: 문제 or 해설 (``prob`` or ``sol``)
* ``<B>``: 이론, 실무, 관측 (각각 ``T``, ``P``, ``O``)
* ``<C>``: 주니어 or 시니어 (``a`` or ``b``)

> 예시 
> * 2010년 IAO 관측 주니어 문제: ``2010_IAO_prob_Oa.tex``
> * 2018년 IAO 이론 시니어 해설: ``2018_IAO_sol_Tb.tex``
